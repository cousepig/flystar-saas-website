GP820 是壹種專業的電源時序分配器，它可將輸入的主電源分配成八路電源輸出，並可以按時間順序依次接通和關閉各路電源，從而有效降低對系統的沖擊，特別適合於大型音響工程使用。

一路主電路輸入，八路受控電源輸出。  
內部微處理器控制，可順序開機，逆序關機，並可多臺級聯使用  
各通道通斷都有 LED 指示，壹目了然。  
標準 2U 機箱，方便安裝。

| 规格参数               |              |
| ---------------------- | ------------ |
| 整機最大輸入功率       | 16KW         |
| 各電源通道最大輸出功率 | 2KW          |
| 延時通斷時間           | 1 秒         |
| 級聯控制               | 有           |
| BYPASS 功能            | 有           |
| ALL OFF 功能           | 有           |
| 尺寸(W×D×H)            | 484×222×88mm |
| 包裝尺寸(W×D×H)        | 555×310×78mm |
| 凈重(kg/Pcs)           | 5kg          |
| 總重(kg/Pcs)           | 6kg          |
